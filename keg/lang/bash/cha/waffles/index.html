Bash MMP: Waffles

Short-Circuit Logic with Butter and Syrup?

Watch the waffles video and write a one line program that combines several command functions using short-circuit logical. Remember to create prompts for user input and to fully validate each response.

Skills and Concepts Covered

The waffles Bash MMP builds on the learning from previous skills and concepts learned from previous MMPs adding the following:

  • Bash while Loops
    • Bash while Statement
    • Bash continue Statement
    • Bash break Statement
  • Bash Flow Control
    • Bash Conditional Statements
      • Bash if Statment
      • Bash Double Bracket [[ ]] Conditions
      • Bash Conditional Operators
    • Short-Circuit Logic
  • Bash Special Variables
  • TODO
  • Subroutines, Procedures, and Functions
  • Procedural Programming Paradigm
  • Functional Programming Paradigm
  • Code Refactoring

Also covered but with less emphasis:

  • POSIX Bracket Conditions [ ]
  • Bash test Command
