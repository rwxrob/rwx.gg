Modern C

Good and Free but Not Beginner Friendly

Modern C is a absolutely free book about the most modern version of the C language. It was last updated on Oct. 10, 2019 and written by a French author Jens Gustedt. Although it is not nearly as beginner-friendly nor does it contain as much content as Head First C it is a useful resource and has been included here as a PDF for reference.
