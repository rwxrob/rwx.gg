Visual Studio Code / VSCode

Visual Studio Code, as Microsoft calls it, or VSCode to the rest of us, is currently the leading GUI editor. VSCode runs on top of the Electron platform created by GitHub to power their Atom editor. Like the rest of the minimal GUI editors to appear VSCode was heavily inspired from Atom and originally Sublime.

Do not confuse Visual Studio Code with Visual Studio.
