Common Programming Language Tasks

Stuff You Usually Need to Learn to Be Productive in Any Language

Most languages all support these tasks, which is why learning a few langauges generally means you can eventually learn any language, which is also why companies generally hire developers as opposed to specific language programmers. Most don’t want to hire a one-trick pony. This ever growing list of tasks can guide you as you assess your own ability to code in any language.
