Bash Language Tasks

HowTo, Recipies, Tutorials

This is a comprehensive list of Bash programming tasks organized by how commonly you might be asked to use them and therefore importance. Master them progressively in order and read about the terms and concepts related to them. Write your own codebook containing code with personal notes as you go. Make sure to exercise your new skills as soon as you learn them by completing and understanding some of these mini projects or others of your own creation.

All the listed tasks could have “from the terminal command line” added to the end. This is assumed.
