Early Motivators for Tech Learning

Keeping It Fun

Having a specific thing in mind makes learning the boring stuff easier because you know what you are eventually going to get out of it. People are motivated by different things. Here are a few examples of project goals that tend to motivate early-stage technologists. Consider picking one and writing it down as part of your learning plan.

  1. Set Up a Minecraft Server
  2. Develop a 2D Web Game
  3. Create a Web Site
  4. Hack and Defend (CTF Games)
  5. Get a Job
