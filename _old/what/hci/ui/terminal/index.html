What is a Terminal User Interface / TUI?

A Way to Use a Program That Usess All of the Terminal Screen

A terminal user interface or TUI is an interface that allows you to control something, usually a computer or device through a full screen terminal, not just a single command line. TUIs are standard for servers that don’t waste resources on a full graphic interface
